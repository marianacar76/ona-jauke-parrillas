# Ona Jauke Parrillas

Sitio web de venta de parrillas artesanales.

## Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox y Grid
- JavaScript
  - Manipulación del DOM
  - LocalStorage (carrito de compras persistente)
  - Fetch API
- Google Fonts
- Formspree

## Secciones
- Inicio
- Productos
- Reseñas
- Contacto

## Funcionalidades
- Catálogo de productos generado dinámicamente
- Botón "Ver más" con detalles de cada parrilla
- Carrito de compras: agregar, eliminar y vaciar productos
- Contador de productos en el carrito, persistente con LocalStorage
- Cálculo automático del total del carrito
- Formulario de contacto funcional
- Diseño responsivo

## Cómo usarlo
1. Cloná este repositorio o descargá los archivos.
2. Abrí `index.html` en tu navegador.
3. No requiere instalación de dependencias.
